# cleantech-grants-tracker

**Live tool:** [moezusb.github.io/cleantech-grants-tracker](https://moezusb.github.io/cleantech-grants-tracker/)

A bilingual (EN/FR) browser-based tracker of federal and provincial funding programs for Canadian cleantech and AI companies. Covers all 10 provinces with verified program details, intake windows, and eligibility criteria.

## What it covers

- **43 programs** spanning federal, provincial, and regional funding sources
- All 10 provinces: BC, AB, SK, MB, ON, QC, NB, NS, PE, NL
- Program types: grants, tax credits, R&D incentives, and conditionally repayable contributions (clearly labeled)

## Features

- Filter by province, status (Open / Rolling / Closing), and category (AI, Cleantech, R&D, Energy, Financing)
- Full-text search across program names, funders, and descriptions
- Bookmark programs you're actively tracking — state persists across sessions
- Expand any card for eligibility details, intake windows, official links, and application notes
- Fully bilingual — toggle between English and French at any time

## Data notes

Programs verified as of Q1 2026. Key context:

| Note | Detail |
|---|---|
| SDTC wound down | Mandate transferred to NRC IRAP Clean Technology (2025–2026) |
| PacifiCan / PrairiesCan | Conditionally repayable — not free grants; labeled accordingly |
| Intake windows | Included where publicly available; verify before applying |

Always confirm program details on official government pages before submitting an application. This tool is for research purposes only.
